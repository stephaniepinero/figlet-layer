# figlet-layer

Se deben instalar las librerias del package.json

``` $ npm install ```

Se debe tener instalado el framework serverless

``` $ npm install -g serverless ```

## Comandos

### Elimiar el layer de AWS

``` $ npm run remove ``` 

### Build del layer de AWS

``` $ npm run build-layer ``` 

### Deploy layer de AWS

``` $ npm run deploy ```
